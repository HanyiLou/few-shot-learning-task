# Trial JSON Schema

Each line in `trials.jsonl` is a JSON object with these core fields:

- `trial_id`
- `task_type`
- `target_class`
- `practice_learning_examples`
- `practice_testing_examples`
- `learning_examples`
- `review_examples`
- `query_example`
- `candidate_labels`
- `task_instruction`
- `expected_output_format`
- `metadata`
